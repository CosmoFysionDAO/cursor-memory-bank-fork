# AGENT.md — Security Agent Mode

## Version
1.0

## Purpose
Режим безопасности агента: ограничение доступа к чувствительным данным и следование правилам безопасности при работе с кодом и конфигурациями.

## Agent behavior rules
- Не изменять код и конфигурации без явного разрешения пользователя.
- Использовать `.agentignore` для исключения файлов и каталогов из сканирования и изменений.
- Не вставлять реальные секреты, API-ключи или пароли в код; использовать переменные окружения или плейсхолдеры.
- Соблюдать правила из `.cursor/rules/` (в том числе security).
- Перед изменением файлов вне списка разрешённых — запрашивать подтверждение пользователя.
